Structured Outputs by Example is a hands-on introduction to working with structured data extraction from LLMs. The site showcases how to use libraries like Instructor and Pydantic to reliably extract structured information from LLM outputs.

Check out the first example or browse the full list of sections below. You may use arrow keys to navigate between examples.

Examples here assume Python >=3.9 and the latest versions of Instructor and Pydantic. Try to upgrade to the latest versions if something isn't working.
